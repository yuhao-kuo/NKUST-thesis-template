## NKUST Thesis LaTeX Version Updates

[繁體中文](update.md) | **English**

When a major update lands after you have started using this template, you can move to the latest version either with `Git` or by hand.
The project separates the writing part from the layout part, so if you use the default layout (without modifying it), most updates will not touch your content.

The following locations contain no user data:

1. ***build***
2. ***Configurations***
3. ***Packages***
4. ***Docker***
5. ***Templates***
6. ***NKUSThesis.cls***
7. ***main.tex***
8. ***titlepage.tex***
9. ***Docs***
10. ***.gitignore***
11. ***README.md***
12. ***Makefile***

The following locations are likely to conflict during an update, so watch for file changes there:

1. ***Chapters***
2. ***Externals***
3. ***Figures***
4. ***Instance***
5. ***References***
6. ***Tables***
7. ***build***
8. ***thesisinfo.tex***
9. ***config.tex***
10. ***Appendices***
